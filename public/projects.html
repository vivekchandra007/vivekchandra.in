Siemens Web Framwork (SWF)

Siemens’ internal (currently) web UI declarative framework on which Siemens industry PLM software products like Active Workspace (TeamCenter), etc are built upon. It is the backbone of Siemens’ Cloud strategy, for building industry scale applications having modern and consistent layout, also not being prone to rapid changes in web technologies, achieved via the framework’s abstraction layer.
